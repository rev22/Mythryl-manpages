! SYNOPSIS

''my''

''my'' [ /runtime system args/ ] [ /options/ ] [ /files/ ]

''my'' -h

! DESCRIPTION

''my'' is a shorthand command for ''mythryld'', which starts an
interactive Mythryl session.

! OPTIONS

Please consult ''mythryld''

! EXAMPLE

''rlwrap my''::
	Starts mythryld with command line history and editing
	provided by ''rlwrap''.  ''rlwrap'' is not provided by the
	mythryl compiler suite	but can be installed separetely.
